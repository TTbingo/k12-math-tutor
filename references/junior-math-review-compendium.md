## 第1章 有理数（七上）

### 核心知识点

1. **正数和负数**：大于0为正，加"-"为负；0既不是正数也不是负数
2. **有理数分类**：整数（正整数、0、负整数）+ 分数（正分数、负分数）
3. **数轴三要素**：原点、正方向、单位长度
4. **相反数**：只有符号不同的两个数，a和-a；0的相反数是0
5. **绝对值**：|a|，数轴上表示a的点到原点的距离
   - a>0 → |a|=a；a=0 → |a|=0；a<0 → |a|=-a
6. **倒数**：乘积为1的两个数；0没有倒数
7. **比较大小**：正数>0>负数；两个负数，绝对值大的反而小
8. **乘方**：aⁿ，a为底数，n为指数；负数的奇次幂为负，偶次幂为正
9. **科学记数法**：a×10ⁿ（1≤|a|<10）；负指数用于小数
10. **近似数**：精确到某一位

### 运算法则

- **加法**：同号相加取同号，绝对值相加；异号取绝对值大的符号，绝对值相减
- **减法**：a-b = a+(-b)
- **乘法**：同号得正，异号得负，绝对值相乘
- **除法**：除以一个数=乘它的倒数
- **混合运算顺序**：乘方→乘除→加减；同级从左到右；有括号先括号

### 课标要点
- 理解有理数意义，能用数轴表示和比较
- 理解相反数、绝对值，掌握|a|含义
- 掌握四则运算及简单混合运算（三步以内）
- 会用科学记数法（含负指数幂）
- 理解运算律并能简化运算

### 常见考点
1. 相反数/绝对值/倒数的求值与辨析
2. 科学记数法表示（含小数）
3. 有理数混合运算
4. 数轴上的距离问题

### 典型易错
- |a| = a 只在 a≥0 成立；a = |a| → a≥0；a = -a → a=0
- 倒数等于本身的数：±1；平方等于本身的数：0,1；立方等于本身的数：0,1,-1
- 科学记数法的小数：0.0000025 = 2.5×10⁻⁶

## 第2章 实数（七下）

### 核心知识点

1. **平方根**：x²=a，则x是a的平方根（a≥0时有两个，互为相反数；0的平方根是0；负数没有）
2. **算术平方根**：正数x满足x²=a，记作√a；0的算术平方根是0
3. **立方根**：x³=a，记作∛a；正数/负数/0都有唯一立方根
4. **无理数**：无限不循环小数
5. **实数分类**：有理数+无理数；实数与数轴上的点一一对应
6. **比较大小**：有理数比较法则对实数同样适用；带根号的无理数比较时"回到根号下"

### 课标要点
- 了解平方根/算术平方根/立方根概念，会用根号表示
- 理解乘方与开方互为逆运算
- 了解无理数和实数概念，实数与数轴一一对应
- 能求实数的相反数与绝对值
- 能估计无理数的大致范围

### 常见考点
1. 算术平方根/平方根/立方根求值
2. 实数分类（有理数 vs 无理数辨析）
3. 实数比较大小与估值（如√10在3和4之间）
4. 实数运算（常与三角函数结合）

### 典型题
- √16的算术平方根是2（陷阱：先算√16=4，再算√4=2）
- 最小的正整数：1；最大的负整数：-1；绝对值最小的数：0

## 第3章 整式（七上/八上）

### 核心知识点

1. **单项式**：数×字母的积；系数+次数（所有字母指数和）
2. **多项式**：几个单项式的和；次数=最高项次数
3. **同类项**：字母相同，相同字母指数也相同
4. **合并同类项**：系数相加，字母及指数不变

### 整式运算公式

| 运算 | 公式 | 说明 |
|------|------|------|
| 同底数幂乘 | aᵐ·aⁿ = aᵐ⁺ⁿ | 底不变，指数加 |
| 幂的乘方 | (aᵐ)ⁿ = aᵐⁿ | 底不变，指数乘 |
| 积的乘方 | (ab)ⁿ = aⁿbⁿ | 各因式分别乘方 |
| 同底数幂除 | aᵐ÷aⁿ = aᵐ⁻ⁿ | a≠0 |
| 零指数 | a⁰ = 1 | a≠0 |
| 平方差 | (a+b)(a-b) = a²-b² | |
| 完全平方 | (a±b)² = a²±2ab+b² | |

### 因式分解方法
1. **提公因式法**：pa+pb+pc = p(a+b+c)
2. **公式法**：a²-b²=(a+b)(a-b)；a²±2ab+b²=(a±b)²

### 去括号/添括号法则
- **同号得正，异号得负**：括号外是正号，去括号后各项不变号；括号外是负号，去括号后各项全变号

### 常见考点
1. 单项式系数/次数判断
2. 同底数幂运算公式辨析
3. 乘法公式（平方差、完全平方）应用
4. 因式分解（提公因式+公式法）
5. 找规律/新定义运算

### 典型易错
- a²·a³ = a⁵ ≠ a⁶
- (x²)⁴ = x⁸ ≠ x⁶
- x²+x² = 2x² ≠ x⁴（合并同类项 vs 同底数幂乘法混淆）

## 第4章 分式（八上）

### 核心知识点

1. **定义**：A/B，B中含有字母且B≠0
2. **基本性质**：分子分母同乘（除）同一个不为0的整式，分式值不变
3. **约分**：约去分子分母公因式 → 最简分式
4. **通分**：异分母化为同分母 → 找最简公分母
5. **运算**：先因式分解，再运算
   - 乘法：分子乘分子，分母乘分母
   - 除法：除式倒置后相乘
   - 加减：同分母分子相加减；异分母先通分
6. **负指数幂**：a⁻ⁿ = 1/aⁿ（a≠0）

### 常见考点
1. 分式有意义的条件（分母≠0）
2. 分式值为0的条件（分子=0且分母≠0）
3. 分式基本性质（x,y同时扩大n倍，值是否变化）
4. 分式化简求值

### 典型易错
- 分式加法：不是分子加分子、分母加分母
- 代入求值前必先化简

## 第5章 二次根式（八下）

### 核心知识点

1. **定义**：√a（a≥0）
2. **基本性质**：(√a)²=a(a≥0)；√a²=|a|（注意a为全体实数时）
3. **乘除**：√a×√b=√ab(a,b≥0)；√a÷√b=√(a/b)(a≥0,b>0)
4. **最简二次根式**：被开方数不含分母、不含能开得尽的因数
5. **加减**：先化最简，再合并同类二次根式

### 常见考点
1. 二次根式有意义的条件（被开方数≥0）
2. 最简二次根式判断
3. 二次根式化简与运算
4. 代数式综合运算（常与分式结合）

### 易错
- √a² = |a| ≠ a（a可能为负）
- 3+√3 ≠ 3√3

## 第6章 一次方程(组)（七上/七下）

### 核心知识点

1. **一元一次方程**：含一个未知数，次数为1，等号两边为整式
2. **等式性质**：a=b → a±c=b±c；a=b → ac=bc（c≠0时a/c=b/c）
3. **解方程步骤**：去分母→去括号→移项→合并同类项→系数化为1
4. **二元一次方程组解法**：
   - **代入消元法**：用含一个未知数的式子表示另一个，代入消元
   - **加减消元法**：两方程相加/减消去一个未知数
5. **应用题步骤**：审题→设未知数→列方程→解方程→答

### 常见考点
1. 方程（组）的解
2. 解一次方程（组）
3. 应用题（行程、工程、利润等）

---

## 第7章 分式方程（八上）

### 核心知识点

1. **定义**：分母中含未知数的方程
2. **解法**：去分母（两边同乘最简公分母）→解整式方程→**检验**（关键步骤！）
3. **应用题**：审→设→列→解→检验→答

### 常见考点
1. 列分式方程
2. 解分式方程（检验增根）
3. 应用题（工作效率、行程等）

## 第8章 一元二次方程（九上）

### 核心知识点

1. **一般形式**：ax²+bx+c=0（a≠0）
2. **四种解法**：
   | 方法 | 适用场景 | 核心操作 |
   |------|---------|---------|
   | 直接开方法 | x²=p 或 (x+n)²=p | 直接开平方 |
   | 配方法 | 通用，但复杂 | 凑完全平方 |
   | 公式法 | 通用 | x=[-b±√(b²-4ac)]/2a |
   | 因式分解法 | 可分解时最快 | 提公因式/平方差 |
3. **判别式 Δ=b²-4ac**：
   - Δ>0：两个不等实根
   - Δ=0：两个相等实根
   - Δ<0：无实根
4. **根与系数关系**：x₁+x₂=-b/a，x₁·x₂=c/a

### 常见考点
1. 一元二次方程定义（a≠0是前提）
2. 判别式应用
3. 解方程
4. 应用题（增长率、面积、传播等）

### 典型易错
- 漏掉 a≠0 的条件
- 用公式法时忘记"先整理成一般形式"
- 应用题增根不检验

## 第9章 不等式(组)（七下）

### 核心知识点

1. **不等式的性质**：
   - 性质1：a>b → a±c > b±c
   - 性质2：a>b，c>0 → ac>bc，a/c>b/c
   - 性质3：a>b，c<0 → ac<bc，a/c<b/c（**不等号方向改变！**）
2. **解集表示**：在数轴上用空心圈（不含等号）或实心点（含等号）
3. **不等式组**：求各不等式解集的公共部分

### 常见考点
1. 不等式性质辨析
2. 解一元一次不等式并在数轴上表示
3. 解不等式组
4. 应用题（最值、方案选择等）

### 典型易错
- 乘除负数时忘记翻转不等号方向

## 第10章 函数及其图象（八上）

### 核心知识点

2. **平面直角坐标系**：x轴（横）、y轴（纵）、原点、四象限
3. **函数定义**：对于x的每个确定值，y有唯一确定值与之对应
4. **函数表示法**：解析式法、列表法、图象法
5. **描点法画图象**：列表→描点→连线

### 常见考点
1. 坐标系中点的坐标
2. 根据象限确定字母取值范围
3. 确定自变量取值范围
4. 根据实际问题判断函数图象

## 第11章 一次函数（八下）

### 核心知识点

1. **定义**：y=kx+b（k≠0）；b=0时为正比例函数y=kx
2. **图象性质**：
   | 条件 | 经过象限 | 增减性 |
   |------|---------|--------|
   | k>0, b>0 | 三、二、一 | y随x增大而增大 |
   | k>0, b<0 | 三、四、一 | y随x增大而增大 |
   | k<0, b>0 | 二、一、四 | y随x增大而减小 |
   | k<0, b<0 | 二、三、四 | y随x增大而减小 |
3. **待定系数法**：设解析式→代入已知点→解方程求系数
4. **与方程/不等式关系**：交点坐标=方程组的解；不等式解集可从图象直接读出

### 常见考点
1. 待定系数法求解析式
2. 图象与性质（k、b符号判断）
3. 与坐标轴交点
4. 一次函数应用题

## 第12章 二次函数（九上）

### 核心知识点

1. **定义**：y=ax²+bx+c（a≠0）
2. **图象**：抛物线；a>0开口向上，a<0开口向下；|a|越大开口越小
3. **五种形式对比**：
   | 形式 | 对称轴 | 顶点 |
   |------|:------:|:----:|
   | y=ax² | y轴(x=0) | (0,0) |
   | y=ax²+k | y轴(x=0) | (0,k) |
   | y=a(x-h)² | x=h | (h,0) |
   | y=a(x-h)²+k | x=h | (h,k) |
   | y=ax²+bx+c | x=-b/2a | (-b/2a, (4ac-b²)/4a) |
4. **增减性**：对称轴左边和右边增减性相反
5. **与一元二次方程的联系**：抛物线与x轴交点横坐标 = 方程ax²+bx+c=0的根
   - Δ>0：两个交点 → 两个不等实根
   - Δ=0：一个交点 → 两个相等实根
   - Δ<0：无交点 → 无实根
6. **平移规则**："左加右减，上加下减"

### 常见考点
1. 二次函数概念（a≠0）
2. 待定系数法求解析式
3. 图象性质（开口方向、对称轴、顶点、增减性）
4. 图象平移
5. 与方程/不等式的综合
6. 应用题（利润最大化、面积最值等）

## 第13章 反比例函数（九下）

### 核心知识点

1. **定义**：y=k/x（k≠0）；也可写作 y=kx⁻¹ 或 xy=k
2. **图象**：双曲线
   - k>0：在一、三象限，每象限内y随x增大而减小
   - k<0：在二、四象限，每象限内y随x增大而增大
3. **k的几何意义**：|k| = 双曲线上任一点向坐标轴作垂线所得矩形面积

### 常见考点
1. 求反比例函数解析式
2. 图象与性质
3. k的几何意义（与面积结合）
4. 与一次函数的综合题

## 第14章 图形初步认识（七上）

### 核心知识点

1. **直线、射线、线段**：
   - 直线：无端点，向两方无限延伸
   - 射线：一个端点，向一方无限延伸
   - 线段：两个端点，能度量
   - 公理：两点确定一条直线；两点之间线段最短
2. **线段中点**：把线段分成相等的两条线段
3. **角**：
   - 分类：锐角(0°~90°)、直角(90°)、钝角(90°~180°)、平角(180°)、周角(360°)
   - 度量：1°=60′，1′=60″
   - 角平分线：角平分线上的点到角两边距离相等
4. **余角与补角**：
   - 余角：和为90°；补角：和为180°
   - 性质：同角（等角）的余角/补角相等

## 第15章 命题、定理与证明（七下）

### 核心知识点

1. **命题**：判断一件事情的语句；由题设和结论组成
2. **真命题/假命题**：题设成立时结论一定成立/不一定成立
3. **定理**：经过推理证实的真命题
4. **逆命题**：题设和结论互换；原命题成立，逆命题不一定成立
5. **证明**：用推理判断命题正确性的过程

## 第16章 相交线与平行线（七下）

### 核心知识点

1. **邻补角与对顶角**：对顶角相等
2. **垂线**：
   - 过一点有且只有一条直线与已知直线垂直
   - 垂线段最短
   - 点到直线的距离 = 垂线段的长度
3. **平行线**：
   - 公理：过直线外一点有且只有一条直线与之平行
   - 传递性：a∥b，b∥c → a∥c
4. **平行线判定与性质**（核心，三句话对应）：

   | 角度关系 | → 线关系（判定） | ← 线关系（性质） |
   |---------|:--------------:|:--------------:|
   | 同位角相等 | 两直线平行 | 两直线平行 |
   | 内错角相等 | 两直线平行 | 两直线平行 |
   | 同旁内角互补 | 两直线平行 | 两直线平行 |

### 常见考点
1. 对顶角/邻补角计算
2. 同位角/内错角/同旁内角识别
3. 平行线判定与性质的综合运用

## 第17章 图形的变换（七下/八上）

### 核心知识点

1. **平移**：图形沿直线方向移动；对应点连线平行（或共线）且相等
   - 坐标变化：(x,y)+右移a→(x+a,y)；+上移a→(x,y+a)
2. **轴对称**：沿直线折叠后重合
   - 对称轴垂直平分对应点连线
   - 线段垂直平分线上的点到两端点距离相等
   - 坐标：(x,y)关于x轴对称→(x,-y)；关于y轴对称→(-x,y)
3. **旋转**：绕定点转动一个角度
   - 对应点到旋转中心距离相等；旋转角相等；图形全等
4. **中心对称**：旋转180°后重合
   - 对称点连线过对称中心且被平分
   - 坐标：(x,y)关于原点对称→(-x,-y)

### 常见考点
1. 轴对称图形 vs 中心对称图形识别
2. 坐标变换
3. 线段垂直平分线性质应用

---

## 第18章 投影与视图（九下）

### 核心知识点

1. **投影**：平行投影（平行光线）、中心投影（同一点发出）、正投影（垂直投影面）
2. **三视图**：主视图（前→后）、俯视图（上→下）、左视图（左→右）
   - 规律：主俯长对正，主左高平齐，俯左宽相等
3. **展开图**：直棱柱、圆锥的侧面展开

## 第19章 三角形（八上）

### 核心知识点

1. **分类**：
   - 按边：不等边、等腰（含等边）
   - 按角：锐角、直角、钝角
2. **三边关系**：两边之和>第三边，两边之差<第三边
3. **内角和外角**：
   - 内角和=180°；直角三角形两锐角互余
   - 外角=不相邻两内角之和；外角和=360°
4. **重要线段**：高、中线、角平分线；三角形具有稳定性
5. **等腰三角形**：
   - 性质：等边对等角；三线合一
   - 判定：等角对等边
6. **等边三角形**：三内角均为60°；有一个角60°的等腰三角形为等边
7. **直角三角形**：
   - 30°角所对直角边=斜边一半
   - 勾股定理：a²+b²=c²
   - 勾股定理逆定理：a²+b²=c² → 直角三角形

### 常见考点
1. 三边关系（能否构成三角形）
2. 内角和/外角计算
3. 等腰三角形分类讨论（角为40°时顶角还是底角？）
4. 勾股定理及其逆定理

### 典型易错
- 等腰三角形已知一角求顶角：必须讨论该角是顶角还是底角
- 已知两边求第三边：直角三角形用勾股定理；一般三角形两边之和>第三边>两边之差
- "两边长3,4"vs"两直角边长3,4"：前者第三边需要讨论

---

## 第20章 全等三角形（八上）

### 核心知识点

1. **性质**：全等三角形对应边相等，对应角相等
2. **五种判定方法**：
   | 判定 | 条件 | 说明 |
   |:----:|------|------|
   | SSS | 三边分别相等 | 基本事实 |
   | SAS | 两边+夹角分别相等 | 基本事实 |
   | ASA | 两角+夹边分别相等 | 基本事实 |
   | AAS | 两角+其中一角对边分别相等 | 推论 |
   | HL | 斜边+一直角边分别相等 | 仅限直角三角形 |

### 常见考点
1. 选择合适的判定方法
2. 全等证明（常见模型：手拉手、一线三等角等）
3. 全等性质与判定的综合

### 📐 辅导要点（给家长/老师）
- 教SSS时强调"稳定性"；教SAS时强调"夹角"不能是"对角"
- 让学生自己画图标注已知条件，再判断能用哪个判定
- 易错：SSA不能判定全等（"两边一角"中的角必须是夹角才是SAS）

---

## 第21章 相似三角形（九下）

### 核心知识点

1. **相似多边形**：角相等，边成比例
2. **相似三角形判定**：
   | 判定 | 条件 |
   |:----:|------|
   | 预备定理 | 平行于三角形一边的直线截另两边 → 相似 |
   | SSS相似 | 三边成比例 |
   | SAS相似 | 两边成比例+夹角相等 |
   | AA | 两角分别相等 |
3. **相似三角形性质**：
   - 对应边成比例，对应角相等
   - 对应高的比 = 对应中线的比 = 对应角平分线的比 = 相似比
   - 周长比 = 相似比
   - **面积比 = 相似比的平方**
4. **位似**：相似+对应点连线交于一点（位似中心）

### 常见考点
1. 相似判定
2. 相似比与面积比的转换
3. 平行线分线段成比例
4. 相似应用（测高、测距等）

---

## 第22章 锐角三角函数（九下）

### 核心知识点

1. **定义**（在Rt△中）：
   - sinA = 对边/斜边
   - cosA = 邻边/斜边
   - tanA = 对边/邻边
2. **特殊角三角函数值**：

   | α | sinα | cosα | tanα |
   |:--:|:----:|:----:|:----:|
   | 30° | 1/2 | √3/2 | √3/3 |
   | 45° | √2/2 | √2/2 | 1 |
   | 60° | √3/2 | 1/2 | √3 |

3. **解直角三角形**：已知两边或一边一角，求其余元素
4. **应用**：仰角/俯角、坡度、方向角

### 常见考点
1. 特殊角三角函数值（常与实数运算结合）
2. 解直角三角形
3. 实际应用题（测高、航海等）

## 第23章 平行四边形（八下）

### 四边形知识链

```
四边形 → 平行四边形 → 矩形
                    → 菱形
                    → 正方形（矩形+菱形）
```

### 各类四边形性质与判定速查

| 四边形 | 定义 | 特有性质 | 判定要点 |
|--------|------|---------|---------|
| 平行四边形 | 两组对边分别平行 | 对边相等、对角相等、对角线互相平分 | ①两组对边分别相等 ②一组对边平行且相等 ③对角线互相平分 |
| 矩形 | 有一个角是直角的▱ | 四个角都是直角、对角线相等 | ①有一个角是直角的▱ ②对角线相等的▱ |
| 菱形 | 有一组邻边相等的▱ | 四边相等、对角线互相垂直且平分对角 | ①一组邻边相等的▱ ②对角线垂直的▱ ③四边相等 |
| 正方形 | 最特殊 | 兼具矩形和菱形所有性质 | 既是矩形又是菱形 |

### 其他要点
- **多边形内角和**：(n-2)×180°；外角和恒为360°
- **三角形中位线定理**：平行于第三边且等于第三边一半

## 第24章 圆（九上）

### 核心知识点

1. **基本概念**：圆心、半径、弦、直径、弧（优弧/劣弧）、等弧
2. **垂径定理**：垂直于弦的直径平分弦，且平分弦所对的两条弧
3. **弧、弦、圆心角关系**：同圆或等圆中，三者有一组量相等，其余两组也相等
4. **圆周角定理**：
   - 圆周角 = 同弧圆心角的一半
   - 同弧/等弧所对圆周角相等
   - 直径所对圆周角=90°；90°圆周角所对弦是直径
   - 圆内接四边形对角互补
5. **点与圆位置关系**：d<r(内)、d=r(上)、d>r(外)；不共线三点确定一个圆
6. **直线与圆位置关系**：d<r(相交)、d=r(相切)、d>r(相离)
   - 切线判定：过半径外端且垂直于半径
   - 切线性质：切线垂直于过切点的半径
   - 切线长定理：从圆外一点引两条切线，切线长相等
7. **三角形与圆**：
   - 外接圆：圆心=外心（垂直平分线交点）
   - 内切圆：圆心=内心（角平分线交点）
8. **弧长与扇形面积**：
   - 弧长：l = nπr/180
   - 扇形面积：S = nπr²/360 = lr/2
   - 圆锥侧面积：πrl

### 常见考点
1. 垂径定理计算
2. 圆周角与圆心角关系
3. 切线证明与计算
4. 与三角形/四边形综合
5. 弧长/扇形面积计算

## 第25章 数据的收集与分析（七下/八下）

### 核心知识点

1. **调查方式**：全面调查（普查）vs 抽样调查
2. **统计术语**：总体、个体、样本、样本容量
3. **统计图表**：条形图、扇形图、折线图、直方图
4. **数据代表**：
   | 指标 | 含义 | 适用场景 |
   |------|------|---------|
   | 平均数 | 数据总和÷个数 | 数据分布较均匀 |
   | 加权平均数 | 各数据×权重之和 | 有重要性差异 |
   | 中位数 | 排序后的中间值 | 有极端值时 |
   | 众数 | 出现次数最多的值 | 分类数据 |
5. **方差**：衡量数据波动程度；方差越大越不稳定

### 常见考点
1. 普查 vs 抽样调查判断
2. 平均数/中位数/众数计算与选择
3. 方差计算与稳定性判断
4. 统计图表的补全与信息提取

## 第26章 概率初步（九上）

### 核心知识点

1. **事件分类**：
   - 必然事件：P=1
   - 不可能事件：P=0
   - 随机事件：0<P<1
2. **概率计算**：P(A) = m/n（m为事件A包含的结果数，n为所有等可能结果总数）
3. **求概率方法**：列表法、树状图法
4. **频率估计概率**：大量重复试验中频率趋于稳定值

### 常见考点
1. 事件类型判断
2. 简单概率计算
3. 列表/树状图求概率
4. 频率估计概率
