# 年级对应知识点速查

> 📚 **完整知识库**：`references/comprehensive-knowledge-base.md`（小学1-6年级）
> 📚 **初中知识库**：`references/junior-math-review-compendium.md`（初中7-9年级，26章含知识点+课标+考点）
> 📚 **高中知识库**：`references/hs-math-review-compendium.md`（高中必修+选必五册，18章深度萃取）
> 📝 **全学段索引**：`references/exam-bank.md`（瘦身心引文件，~120行），指向各学段参考资料
> 📝 **经典案例库**：`references/case-studies.md`（12个经典启发案例）

| 年级 | 核心知识点 | 关键算理 |
|------|-----------|---------|
| 一年级 | 20以内加减法、认识10 | 凑十法（理解数的分拆，而不是死记） |
| 二年级 | 表内乘除法、100以内加减 | 乘法本质是"相同加数的加法"，除法是"平均分" |
| 三年级 | 多位数乘除法、分数初步 | 竖式的位值原理，分数是"整体的几份" |
| 四年级 | 大数运算、小数加减 | 小数点对齐 = 位值对齐 |
| 五年级 | 分数四则、方程入门 | 分数乘除的图形理解，方程是"等量关系" |
| 六年级 | 比例、百分数、统计 | 比例是对应关系，百分数是特殊分数 |
| 初一（七年级） | 有理数运算、整式加减、一元一次方程、实数、平面直角坐标系、二元一次方程组、不等式 | 负数扩展数轴，方程是"求未知量的相等关系"，坐标系是"位置的语言" |
| 初二（八年级） | 三角形、全等三角形(SSS/SAS/ASA/AAS/HL)、轴对称、整式乘除与因式分解、分式、二次根式、勾股定理、平行四边形(矩形/菱形/正方形)、一次函数 | 全等是"变换不变性"，因式分解是"乘法公式逆运算"，函数是"两个变量的关系" |
| 初三（九年级） | 一元二次方程(四种解法+判别式)、二次函数(五种形式+图象性质)、旋转、圆(垂径/圆周角/切线)、反比例函数、相似三角形、锐角三角函数(sin/cos/tan)、投影与视图、概率初步 | 二次函数是"变化率的可视化"，圆是"到定点等距点的集合"，三角函数是"相似三角形的应用" |
| 高一（必修一） | 集合与逻辑、一元二次函数方程不等式、函数概念与性质、指数对数函数、三角函数 | 集合是"数学语言"，函数是"集合间的对应"，三角函数完成"静态角→动态角→周期模型"三次跃迁 |
| 高一（必修二） | 平面向量、复数、立体几何初步、统计、概率 | 向量是"几何+代数的桥梁"，复数完成"数系封闭"，概率是"不确定性的量化" |
| 高二（选必一） | 空间向量与立体几何、直线和圆的方程、圆锥曲线（椭圆·双曲线·抛物线） | 坐标法三级跳：二维→三维→曲线，圆锥曲线是"坐标法的巅峰应用" |
| 高二（选必二） | 数列（等差·等比·数学归纳法）、一元函数导数及其应用 | 数列是"离散型函数"，导数是"瞬时变化率"，两章合起来掌握分析函数的完整武器 |
| 高二/高三（选必三） | 计数原理（排列组合·二项式定理）、随机变量及其分布、成对数据的统计分析 | 计数→概率→统计推断三层递进，完成"不确定性数学"闭环 |
| 高三（选修+复习） | 复数深化、数学建模、专题深化 | 复数是"实数的二维扩展"，数学建模是"用数学语言描述现实" |
